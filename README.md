# 🛠️ 개발 보조 도구 포털 (Developer Assistant Tools)

효율적인 개발 업무를 위해 반복적인 작업들을 자동화하고 시각화해주는 경량 도구 모음입니다. 별도의 서버 설치 없이 웹 브라우저에서 바로 실행 가능합니다.

## 🚀 빠른 시작

1.  `index.html` 파일을 웹 브라우저로 엽니다.
2.  대시보드에서 원하는 도구를 선택합니다.
3.  상단 내비게이션 바를 통해 다른 도구로 즉시 전환할 수 있습니다.

## 🧰 주요 도구 (현재 활성화됨)

### ✅ TODO 관리 (`todo-manager.html`)
- 월간 달력 기반의 일정 및 태스크 관리 시스템입니다.
- IndexedDB를 사용한 로컬 데이터 영구 보존 및 백업/복구 기능을 제공합니다.
- **연속(기간) 일정**: 시작일~종료일을 하나의 일정으로 등록(달력에 기간만큼 표시, 단일 레코드로 완료/수정/삭제). 등록 모달의 토글로 "연속 일정 / 매일 개별"을 선택합니다.
- 일정 더블클릭 또는 "🔍 크게 보기"로 상세 보기(텍스트/마크다운).
- **구글 캘린더 연동(ICS/MD 내보내기)**: 종일/기간 이벤트로 내보내며, **완료한 일정은 `STATUS:CANCELLED`로 표기**되어 같은 캘린더에 다시 가져오면 제거됩니다(완료 일정 알림 정리용, 전용 캘린더 사용 권장).

### 📝 마크다운 (`md-viewer.html`)
- 실시간 마크다운 렌더링 및 미리보기를 제공합니다.
- **텍스트 추출 모드**: 마크다운 문법을 제거한 순수 텍스트 + 단어/글자/공백제외 수를 보여주고 복사·다운로드합니다. (기존 '마크다운 변환' 도구를 흡수)
- **주요 특징**: 라이트/다크 모드 전환, 실시간 데바운스(Debounce) 렌더링, 코드 구문 강조(Highlight.js).
- 오프라인 사용을 위해 필수 라이브러리가 로컬(`tools/lib/`)에 내장되어 있습니다.

### 🔗 URI 도구 (`uri-tool.html`)
- URL 인코딩, 디코딩 및 쿼리 파라미터를 분석합니다.
- 디코딩된 결과에서 `?key=value` 형태의 파라미터를 자동으로 추출하여 표 형식으로 제공합니다.
- 인코딩 방식 선택(Component / 전체 URI), 디코딩 시 `+`→공백(폼 인코딩), 파일 드래그앤드롭 지원.

### 🧩 JSON / XML 도구 (`json-tool.html`)
- **JSON**: 포맷(정렬)·압축·실시간 검증, 접기/펼치기 + **검색·경로복사** 트리 뷰, **키 정렬**.
- **XML**: DOMParser 기반 **정렬·유효성 검사**, escape/unescape. (기존 XML 변환 도구를 흡수)
- **변환**: JSON ↔ XML, JSON → CSV, escape 문자열 풀기(로그용).
- 파일 드래그앤드롭, 입력 자동 저장(localStorage), Ctrl+Enter 정렬 지원.

### 🔍 텍스트 비교 (`diff-tool.html`)
- 두 텍스트/코드를 LCS 알고리즘으로 **줄 단위 비교**하여 추가·삭제를 색상으로 강조합니다.
- **줄 내 단어 단위 강조**, 끝 공백/대소문자/빈 줄/공백 전체 무시 옵션, 자동 재비교.
- 결과 복사(+/- 형식), 양쪽 파일 드래그앤드롭 지원.

### ⏱️ 타임스탬프 / Cron (`time-tool.html`)
- Epoch(초/밀리초 자동 인식) ↔ 날짜/타임존/ISO 상호 변환 및 상대 시간 표시.
- **세계 시간**(여러 타임존 동시 표시), **날짜 계산기**(차이·가감·D-day).
- **Cron 표현식 해석**(5필드/6필드[초 포함, Spring]/`@daily` 등 별칭)과 **다음 실행 시각** 미리보기.

> 🌙 **전역 다크모드**: 상단 내비게이션 바(또는 대시보드 우측 상단)의 토글로 포털 전체 테마를 전환합니다. 상태는 `localStorage`에 저장되며, `postMessage`로 iframe 내 도구까지 실시간 동기화됩니다.

## 🧪 실험적/임시 비활성 도구
*현재 주석 처리되어 대시보드에는 보이지 않지만, 코드는 유지되고 있는 기능들입니다.*

- **패치 내역 생성** (`svn-log-converter.html`): SVN 로그를 기반으로 배포용 패치 파일 목록을 생성합니다.
- **SQL문 변환**: 로그의 SQL 플레이스홀더(`?`)를 실제 값으로 치환합니다.
- **SVN 머지 도구**: 브랜치 간 코드 머지 작업을 시각적으로 보조합니다.
- **Config 설정**: 프로젝트 환경 설정 파일을 시각적으로 편집합니다.
- **Classpath/Maven 설정**: Eclipse 및 Maven 설정 파일을 효율적으로 관리합니다.

## 🛠️ 기술 스택 및 라이브러리
- **Core**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Library & Utils (Internalized)**: 
  - `marked.js`: 마크다운 파싱
  - `highlight.js`: 코드 구문 강조
  - `github-markdown-css`: GitHub 스타일 렌더링
  - `db-manager.js`: IndexedDB 공통 관리 모듈 (자체 개발)
  - `shared-utils.js`: 공통 유틸리티 함수 모음 (자체 개발)
  - `portal-theme.js`: 포털 전역 라이트/다크 테마 관리 모듈 (자체 개발)
- **Storage**: 브라우저 `IndexedDB`, `LocalStorage`

## 📁 디렉토리 구조
- `index.html`: 도구 포털 메인 대시보드
- `tool.html`: 개별 도구를 실행하는 프레임워크 (상단 내비게이션 포함)
- `tools/`: 개별 도구 소스 코드 (`.html`)
- `tools/lib/`: 외부 라이브러리 및 자체 개발 공통 JS 파일 보관소
  - 외부 라이브러리: `marked.min.js`, `highlight.min.js` 등
  - 공통 모듈: `db-manager.js`, `shared-utils.js`
